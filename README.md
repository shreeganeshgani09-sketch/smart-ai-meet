# Smart AI Meet

## Overview

Smart AI Meet is an AI-powered online meeting platform designed to provide secure, intelligent, and efficient virtual meetings. The system combines video conferencing, biometric verification, AI-based face monitoring, ID authentication, and chatbot assistance to improve meeting security and user experience.

## Features

### Secure User Authentication
- User registration and login
- Identity verification using uploaded documents
- Secure access control

### Biometric Face Verification
- Face recognition before joining a meeting
- Verification against registered biometric data
- Prevention of unauthorized access

### Continuous Face Monitoring
- Real-time face tracking during meetings
- Detection of face mismatches and suspicious activity
- Enhanced meeting security

### AI Chatbot Assistant
- Answers user queries during meetings
- Provides meeting-related assistance
- Improves user engagement and support

### Meeting Management
- Create meetings
- Join meetings using meeting IDs
- Schedule and manage meetings
- Participant management

### Real-Time Communication
- Video conferencing
- Audio communication
- Screen sharing support
- Live interaction among participants

## System Architecture

1. User registers and uploads identity information.
2. System performs biometric face verification.
3. Verified users can create or join meetings.
4. AI continuously monitors participants during meetings.
5. AI chatbot provides assistance when required.
6. Meeting data is securely managed by the backend server.

## Installation

### Clone Repository

```bash
git clone https://github.com/your-username/smart-ai-meet.git
```

### Navigate to Project Folder

```bash
cd smart-ai-meet
```

### Install Dependencies

```bash
npm install
```

### Run Frontend

```bash
npm start
```

### Run Backend

```bash
node server.js
```
